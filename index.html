Important FAQ

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface that represents HTML or XML documents as a tree of nodes. Each element and attribute is a node, allowing dynamic manipulation of content, structure, and style using languages like JavaScript for interactive web development.

How do you select on element from the DOM ?

To select an element from the DOM, use document.getElementById(id) for a unique ID, document.querySelector(selector) for the first matching element, or document.querySelectorAll(selector) for all matches. Additionally, document.getElementsByClassName(className) selects elements by class name, and document.getElementsByTagName(tagName) selects elements by tag name.

What is event delegation in the context of the DOM, and why is it useful ?

Event delegation involves attaching a single event listener to a parent element to handle events from its children via event bubbling. It's useful for improved performance, reduced memory usage, and handling dynamically added elements, as you don't need to attach individual event listeners to each child element.

How do you manipulate on element's attributes and styles using the DOM ?

Attributes can be manipulated using setAttribute() to set or update, and removeAttribute() to delete. Styles are modified with element.style.property. CSS classes are managed dynamically using classList.add() or classList.remove().
